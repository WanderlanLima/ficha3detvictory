# 3D&T Victory Builder

Uma aplicação web para criar, salvar e compartilhar fichas de personagens de 3D&T Victory no formato de cartas TCG.

## Funcionalidades

- **Criação de Fichas**: Edite estatísticas, vantagens, desvantagens e imagem.
- **Visualização TCG**: Veja sua ficha como um card colecionável.
- **Salvar**: As fichas ficam salvas no seu navegador (Local Storage).
- **Compartilhar**: Exporte o card como imagem para compartilhar.
- **PWA**: Instale o app no seu celular (Android/iOS) direto pelo navegador.

## Como Usar

1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. Abra o terminal na pasta do projeto.
3. Instale as dependências:
   ```bash
   npm install
   ```
4. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
5. Abra o link exibido no navegador (geralmente `http://localhost:5173`).

## Instalação no Celular

- Acesse a aplicação pelo navegador do celular (na mesma rede Wi-Fi).
- Toque em "Adicionar à Tela Inicial" ou "Instalar Aplicativo".

## Nota sobre o 3D&T Victory

Este aplicativo fornece a estrutura para criar fichas baseadas nas regras. Consulte os livros PDF oficiais na pasta anterior para detalhes sobre custos de vantagens e atributos.
